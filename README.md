## Projeto Extensionista: Sistema de Automação de Pedidos - Ateliê Jenifer Gama

Este repositório contém o projeto de extensão universitária interdisciplinar desenvolvido com o objetivo de conectar o conhecimento acadêmico às necessidades reais da comunidade, apoiando o comércio local.

Sobre o Projeto

O **Ateliê Jenifer Gama Confeitaria** é uma confeitaria de bairro especializada na produção e entrega de doces por encomenda para outros comércios (B2B - CNPJ). Atualmente, o fluxo de pedidos é descentralizado: os clientes realizam os pedidos via WhatsApp e a gestão/anotação é feita manualmente em papel. Esse processo analógico gera problemas de organização, risco de perda de prazos e dificuldade em mensurar os dados do negócio.

**Solução:** Desenvolver uma plataforma web para automatizar o recebimento de pedidos, organizar o fluxo de produção e centralizar o gerenciamento de clientes corporativos.

---

## Integração Disciplinar 
O projeto é o resultado da sinergia entre quatro disciplinas fundamentais do curso:

*   **Introdução à Administração (ADM):** Responsável pelo mapeamento do modelo de negócios (B2B), entendimento das dores do cliente, análise de viabilidade e melhoria do fluxo de processos (do pedido à entrega).
*   **Interação Humano-Computador (IHC):** Utilizada no design da experiência do usuário (UX) e interface (UI). Garantiu que a plataforma fosse intuitiva tanto para os donos de comércios que compram, quanto para a equipe da confeitaria que gerencia os pedidos.
*   **Desenvolvimento Web:** Implementação prática da interface (Frontend) e das regras de negócio/rotas do sistema (Backend).
*   **Banco de Dados (BD):** Modelagem e implementação do banco de dados relacional para armazenar com segurança as informações de clientes (CNPJ), produtos, relatórios de vendas e o status dos pedidos.

---

## Funcionalidades Principais

### Para o Cliente (Comércios/CNPJ ou CPF)
*   Cadastro de empresa (CNPJ) ou Pessoa Física (CPF) e histórico de compras.
*   Catálogo digital de produtos atualizado.
*   Carrinho de compras e envio de pedidos de forma 100% digital.

### Para o Administrador (Ateliê Jenifer Gama)
*   Painel de controle (Dashboard) com pedidos do dia/semana.
*   Alteração de status do pedido (Pendente, Em Produção, Saiu para Entrega, Entregue).
